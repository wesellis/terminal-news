## [0.1.0] - TBD

### MVP Release

#### Added
- Terminal UI with Bubbletea framework
- Hot news feed with community voting
- Article browsing and reading
- User authentication (register/login)
- Vote system (open, like, dislike)
- NOAA weather widget
- Keyboard navigation
- Basic news aggregation from RSS feeds
- PostgreSQL database backend
- Redis caching layer
- RESTful API server
- JWT-based authentication

#### Features
- Browse hot articles
- Vote on articles (like/dislike)
- View article summaries
- Open articles in browser
- Real-time weather display
- User profiles
- Responsive terminal UI

#### Technical
- Go backend with Chi router
- PostgreSQL for data storage
- Redis for caching
- Docker support
- Database migrations
- API documentation
